| gitmoji        | type     | instructions                                       |

| 🐛 :bug:       | fix      | 버그 수정                                          |
| 📝 :memo:      | docs     | 문서 수정                                          |
| ✅             | test     | 테스트 코드 추가                                   |

| 🚚  :truck:    | rename   | 파일 혹은 폴더명을 수정하거나 옮기는 작업만인 경우 |
| 🔥 :fire:      | remove   | 파일을 삭제하는 작업만 수행한 경우                 |
